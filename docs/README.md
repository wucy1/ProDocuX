## 💡 功能特色

### 智能文檔轉換
- 支援多種文檔格式（PDF、Word、DOCX）
- AI驅動的智能提取
- 結構化資料輸出

### 工作流程管理
- 創建和命名專門的工作
- 智能Profile推薦
- 工作歷史追蹤

### AI學習功能
- Word文檔學習模式
- JSON資料學習模式
- 持續優化提取準確度

### 多AI模型支援
- OpenAI GPT系列
- Claude系列
- Gemini系列
- Grok系列
