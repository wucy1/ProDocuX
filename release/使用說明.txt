# ProDocuX RC1 使用說明

## 🚀 快速開始

### 1. 啟動應用
- 雙擊 `ProDocuX.exe` 啟動程式
- 程式會自動開啟瀏覽器並導向設置頁面

### 2. 完成初始設置
- 選擇AI提供者（OpenAI、Claude、Gemini或Grok）
- 輸入對應的API金鑰
- 選擇工作目錄位置
- 選擇要創建的桌面快捷方式

### 3. 創建第一個工作流程

**重要：ProDocuX 的核心是工作流程，必須先創建工作才能處理文檔！**

#### 步驟1：填寫基本資訊
1. 點擊「新建工作」按鈕
2. 填寫工作資訊：
   - 工作名稱：如「3M產品PIF轉換」
   - 工作描述：詳細描述工作目的和範圍
   - 文檔類型：選擇文檔類型（PIF、MSDS、合約等）
   - 品牌/公司：填寫品牌名稱（可選）

#### 步驟2：上傳輸出模板
1. 點擊「選擇模板檔案」
2. 上傳您的輸出格式模板（Word、Excel等）

#### 步驟3：生成Profile（文檔提取配置）
1. 點擊「生成Profile提示詞」
2. 複製提示詞到您的AI工具（ChatGPT、Claude等）
3. 將AI返回的JSON配置貼回系統
4. 點擊「驗證Profile」確認格式正確

#### 步驟4：生成提示詞（文檔處理指令）
1. 點擊「生成提示詞模板」
2. 複製提示詞到您的AI工具
3. 將AI返回的提示詞貼回系統
4. 點擊「驗證提示詞」確認內容正確

#### 步驟5：完成工作創建
1. 點擊「創建工作」
2. 系統會自動選擇新創建的工作

### 4. 開始處理文檔
1. 選擇要使用的工作
2. 上傳要處理的PDF或Word檔案
3. 點擊「開始處理」按鈕
4. 下載處理結果

## 📝 注意事項

- **核心概念**：ProDocuX 是工作流程驅動的系統
- **必須步驟**：先創建工作，再處理文檔
- **AI工具**：需要自己的AI帳戶來生成Profile和提示詞
- **模板上傳**：必須上傳輸出格式模板
- **持續學習**：每個工作都會獨立學習和優化
- 建議定期備份工作目錄中的重要檔案
